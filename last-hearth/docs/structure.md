# Структура проекта Last Hearth

## Основные файлы

```
last-hearth/
├── README.md                 # Общее описание проекта и первый этап
├── index.html               # Главная HTML страница
├── package.json             # Конфигурация npm-пакета
├── tsconfig.json            # Конфигурация TypeScript
├── public/                  # Публичные ресурсы
│   └── assets/              # Изображения для игры
│       ├── player.svg       # Изображение игрока (стрелок)
│       ├── monster.svg      # Изображение монстра
│       └── bullet.svg       # Изображение пули
└── src/                     # Исходный код игры
    └── main.ts              # Основной файл Phaser-приложения
```

## Описание компонентов

### main.ts
Основной файл игры, содержащий:

- Конфигурацию Phaser
- Состояния игры (preload, create, update)
- Логику движения монстров
- Систему стрельбы
- Систему HP таверны
- Экран рассвета
- Систему сохранения/загрузки

### Сцены и объекты

#### preload()
- Загружает изображения для спрайтов

#### create()
- Создает игрока, монстров и пули
- Устанавливает физику
- Создает текстовые элементы интерфейса
- Загружает сохраненный прогресс

#### update()
- Обрабатывает управление игроком
- Обрабатывает стрельбу
- Обновляет позиции монстров
- Удаляет пули за пределами экрана

## Игровая механика

### Основной цикл
1. Игрок защищает таверну, стреляя по монстрам
2. При попадании монстра в игрока, HP таверны уменьшается
3. При уничтожении всех монстров показывается экран рассвета
4. При достижении 0 HP игра заканчивается
5. Игра сохраняет прогресс в localStorage
